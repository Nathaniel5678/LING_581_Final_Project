# LING_581_Final_Project
This is my final project that performs sentiment analysis on a dataset of Letterboxd movie reviews.

This study expands on previous research in sentiment analysis of movie reviews, specifically those from Letterboxd, by using a dataset that includes both star ratings and the reviews themselves. This improves the analysis by adding in evaluation metrics to themodel, such as accuracy, precision, recall, and F1 score. The study compared the accuracy
of the model’s predictions on reviews filled with sarcasm versus more traditional reviews. While baseline accuracies of more general models were lower at around 0.63, finetuning the model with the Letterboxd data improved the total accuracy of the model by around 22 percentage points. The sarcastic review accuracy performance rose from 0.5 to 0.85, showing that finetuning on a model of domain specific reviews can increase the accuracy of the model dramatically.
